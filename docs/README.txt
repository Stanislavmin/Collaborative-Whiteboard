Place your team contract and design documents in this directory.

The files in this directory should be:
- team-contract.pdf
- design-milestone.pdf
- design-revised.pdf
- design-diagram.pdf
- screenshot.png

If you add any other documents to this directory, please add the filenames to
the above list.

To start the server you use the command "Server [--PORT]" which allows the optional argument 
PORT, specifying the port the server will be listening on.

To start the client you use the command Client. There will be a popup where you can choose the 
server ip and port address. These values are set to 127.0.0.1 and 4444 by default.
